GigFin : Money Made Simple for the Self‑Made

UI/UX documentation for a financial & networking app that helps freelancers manage irregular income, plan for taxes, save effectively and showcase their services.

Figma

1. Overview

GigFin is a mobile application that empowers freelancers and gig workers to manage irregular income with ease. It combines financial planning tools with a lightweight networking platform to help users track earnings, plan for taxes, save effectively, and discover or offer freelance services.

Platform

Mobile (React Native)

Project Type

Financial & Networking App for Freelancers

2. Target Audience

  • Freelancers & gig workers
  • Informal workers & side‑hustlers
  • People hiring freelance talent (entrepreneurs, SMEs)
  • Primarily ages 18–40
  • Mobile‑first users (progressive access to web)

3. Design Goals

  • Simplicity: Minimal layouts for quick access
  • Consistency: Shared colours & type across platforms
  • Empowerment: Copy & visuals that signal control
  • Accessibility: Contrast, readable fonts, responsive

7. User Flows

New User

  1. Splash with logo
  2. Sign up / Login
  3. Optional: Create Gig Profile
  4. Dashboard: income/expense, savings, discovery

Returning User

  1. Open to dashboard
  2. Add income/expense
  3. View reports or savings insights
  4. Update gig profile

Hire Journey

  1. Browse/Search gig profiles
  2. Tap to view details
  3. Contact via email/WhatsApp

8. UX Considerations

  • Whitespace to reduce cognitive load
  • CTA hierarchy: gold primary, teal secondary
  • Passive UI (use without full profile)
  • Gentle microinteractions

9. Branding Assets

  • Logo: Wallet + leaf sprout (dark teal, teal, yellow)
  • Slogan: “Money Made Simple for the Self-Made”
  • Iconography: Minimal, outlined
